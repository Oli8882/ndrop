# NDrop

An Android application featuring NFC and location-based functionalities.

## Features
- **NFC Integration**: Seamlessly read and write NFC tags.
- **Location Services**: Advanced location tracking and geofencing capabilities.
- **Home Screen Widgets**: Includes custom widgets (e.g., Parking Widget) for quick access.
- **Modern UI**: Built using modern Android architecture and Jetpack Compose.

## Getting Started

### Prerequisites
- **Android Studio** (latest stable version recommended)
- An Android device or emulator for testing NFC and location features.

### Installation & Building
1. Clone the repository:
   ```bash
   git clone https://github.com/YourUsername/ndrop.git
   ```
2. Open the project in Android Studio.
3. Wait for the Gradle sync to complete.
4. Click **Run** to build and launch the app on your device or emulator.

## Tech Stack
- Kotlin
- Jetpack Compose
- Android NFC API
- Android Location & Geofencing APIs
